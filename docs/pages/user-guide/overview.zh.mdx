# 总览

CPLib 为四种环境提供帮助：

- Checker，也就是自定义判分器．如果你的题目需要细化得分需求，可以使用它实现．
- Interactor，也就是交互库．你可以使用它实现交互题的库．
- Validator，也就是数据校验器．在 **严格模式** 下，你可以将用户提供的数据识别精度达到每一个字符．
- Generator，也就是数据生成器．用于批量生成数据．

本文档将会以结合实例的方式讲解 CPLib 的用法．另外，不论你之前是否有使用过 Testlib 或其他定位类似的库进行开发的经验，都建议你从 [Checker](/user-guide/checker) 部分开始阅读．
